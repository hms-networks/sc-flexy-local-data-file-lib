Utility classes to manage the creation (and eventual deletion) of local data files which contain the data points and alarms processed by the Ewon Flexy. This library is dependent on the HMS Networks MU Americas Solution Center library for data points, found at https://github.com/hms-networks/sc-flexy-datapoint-lib. This library is dependent on the HMS Networks MU Americas Solution Center library for tag info, found at https://github.com/hms-networks/sc-flexy-tag-info-lib. This library is dependent on the HMS Networks MU Americas Solution Center library for file utilities, found at https://github.com/hms-networks/sc-flexy-file-util-lib. This library is dependent on the HMS Networks MU Americas Solution Center library for logging, found at https://github.com/hms-networks/sc-flexy-logger-lib.
